Las entradas del blog soportan [características de Docusaurus Markdown](https://docusaurus.io/docs/markdown-features), como [MDX](https://mdxjs.com/).

:::consejo

Utiliza la potencia de React para crear entradas de blog interactivas.

```js
<button onClick={() => alert('¡botón pulsado!')}>¡Pulsa sobre mí!</button>
```

<button onClick={() => alert('¡botón pulsado!')}>¡Hazme clic!</button>
