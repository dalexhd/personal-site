# Características de Markdown

Docusaurus soporta **[Markdown](https://daringfireball.net/projects/markdown/syntax)** y algunas **características adicionales**.

## Portada

Los documentos Markdown tienen metadatos en la parte superior llamados [Front Matter](https://jekyllrb.com/docs/front-matter/):

```text title="my-doc.md"
// highlight-start
---
id: my-doc-id
title: El título de mi documento
description: La descripción de mi documento
slug: /my-custom-url
---
// highlight-end

## Markdown heading

Markdown text with [links](./hello.md)
```

## Enlaces

Se admiten enlaces regulares de Markdown, utilizando rutas url o rutas de archivo relativas.

```md
Veamos cómo [Crear una página](/crear-una-página).
```

```md
Veamos cómo [Crear una página](./create-a-page.md).
```

**Resultado:** Veamos cómo [crear una página](./create-a-page.md).

## Imágenes

Las imágenes normales de Markdown son compatibles.

Añade una imagen en `static/img/docusaurus.png` y muéstrala en Markdown:

```md
[Logotipo del Docusaurus](/img/docusaurus.png)
```

![Logotipo del Docusaurus](/img/docusaurus.png)

## Bloques de código

Los bloques de código Markdown son compatibles con el resaltado de sintaxis.

    ```jsx title="src/components/HelloDocusaurus.js"
    function HolaDocusaurus() {
        return (
            <h1>¡Hola, Docusaurus!</h1>
        )
}
    ```

```jsx title="src/components/HelloDocusaurus.js"
function HolaDocusaurus() {
  return <h1>¡Hola, Docusaurus!</h1>;
}
```

## Admoniciones

Docusaurus tiene una sintaxis especial para crear advertencias y avisos:

    :::consejo Mi consejo
    
    Utiliza esta impresionante opción de función
    
    :::
    
    :::peligro Tenga cuidado
    
    Esta acción es peligrosa
    
    :::

:::consejo Mi consejo

Utiliza esta increíble opción

:::

:::peligro Cuídate

Esta acción es peligrosa

:::

## Componentes MDX y React

[MDX](https://mdxjs.com/) puede hacer que tu documentación sea más **interactiva** y permite utilizar cualquier componente de **React dentro de Markdown**:

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '20px',
      color: '#fff',
      padding: '10px',
      cursor: 'pointer',
    }}
    onClick={() => {
      alert(`Has hecho clic en el color ${color} con etiqueta ${children}`)
    }}>
    {children}
  </span>
);

¡Esto es <Highlight color="#25c2a0">verde Docusaurus</Highlight>!

¡Esto es <Highlight color="#1877F2">azul de Facebook</Highlight>!
```
